## 4. Известные ограничения MVP

- **HTTP transport client** — заглушка (post-MVP). Клиент использует gRPC.
- **Ротация ключей** — ручная процедура через вызов из кода, нет admin endpoint/CLI-команды.
- **Desktop/Web-клиенты** — вне MVP.
- **OTP/TOTP** — вне MVP.
- **TUI-интерфейс** — вне MVP.
- **Импорт/экспорт данных** — вне MVP.

## 5. Итоговое решение

**Готово к выпуску MVP.**

Все критерии качества, безопасности и документации выполнены в рамках заявленного scope. Ограничения MVP явно зафиксированы и не блокируют выпуск.
